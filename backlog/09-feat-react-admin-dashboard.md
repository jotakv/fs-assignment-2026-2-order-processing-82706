# 9) `feat/09-react-admin-dashboard`

## User story

**As an administrator, I want a separate dashboard to monitor and filter orders so that I can see operational data beyond the customer portal.**

## Descripción

Segundo frontend obligatorio.

## Criterios de aceptación

- listado de pedidos
- filtros por estado
- detalle
- payment status
- inventory result
- failed orders
- consume API

## Tareas técnicas

- app React
- tabla de pedidos
- filtros
- detalle con datos operativos
- pantalla de fallidos

## Prompt para Codex

```text
Work only on branch feat/09-react-admin-dashboard.

Goal:
Create or complete the separate React admin dashboard for operational monitoring of distributed orders.

Requirements:

  Orders Dashboard, Orders Table, Order Details, Failed Orders

- Do not redesign the whole backend

Deliver:
- code changes
```
